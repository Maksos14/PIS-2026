5.2. Обработка исключительных ситуаций
Реализовано стратегий обработки: 7
Примеры:

Исключительная ситуация 1: Воронка этапов не настроена
•	Условие: Отсутствует конфигурация воронки этапов у пользователя
•	Обнаружение: Stage Service возвращает ошибку stage_creation_failed
•	Реакция: ROLLBACK транзакции, логирование критической ошибки
•	Компенсация: Полный откат транзакции
•	Уведомление: "Не удалось добавить отклик. Обратитесь в поддержку"

Исключительная ситуация 2: SMS сервис недоступен
•	Условие: Внешний SMS-шлюз не отвечает на запросы
•	Обнаружение: TimeoutException через 10 секунд ожидания
•	Реакция: Логирование ошибки, создание задачи в очереди повторных попыток
•	Компенсация: Retry-механизм (5 попыток с экспоненциальной задержкой)
•	Уведомление: "Отклик добавлен! SMS будет отправлено позже"

Исключительная ситуация 3: Дубликат отклика
•	Условие: Пользователь уже откликался на эту вакансию в течение 30 дней
•	Обнаружение: SQL запрос находит существующую запись с таким же (user_id, company_id, vacancy_title)
•	Реакция: Отмена операции, показ предупреждения
•	Компенсация: Не требуется (данные не изменялись)
•	Уведомление: "Вы уже откликались на эту вакансию N дней назад"

Исключительная ситуация 4: Таймаут базы данных
•	Условие: Запрос к БД выполняется дольше 5 секунд
•	Обнаружение: Драйвер БД выбрасывает SQLTimeoutException
•	Реакция: ROLLBACK транзакции, возврат HTTP 503
•	Компенсация: Полный откат транзакции
•	Уведомление: "Сервис временно недоступен. Попробуйте позже"

Исключительная ситуация 5: Дата в будущем
•	Условие: Пользователь указал дату отклика позже текущей
•	Обнаружение: Валидация responseDate > currentDate на уровне API
•	Реакция: Блокировка отправки формы, показ предупреждения
•	Компенсация: Не требуется
•	Уведомление: "Дата отклика не может быть в будущем. Использовать текущую?"

Исключительная ситуация 6: Компания не найдена
•	Условие: Введённое название компании отсутствует в справочнике
•	Обнаружение: SELECT в таблицу companies возвращает 0 записей
•	Реакция: Открытие формы добавления новой компании
•	Компенсация: Создание компании в отдельной транзакции
•	Уведомление: "Компания не найдена. Добавить новую?"

Исключительная ситуация 7: Ошибка создания напоминания
•	Условие: Reminder Service недоступен или возвращает 500
•	Обнаружение: HTTP 500 при вызове /api/reminders
•	Реакция: Логирование ошибки, создание фонового Job'а
•	Компенсация: Асинхронное создание напоминания позже
•	Уведомление: Без уведомления (операция выполнена успешно для пользователя)
